# Pokemon CRUD Backend (TypeScript)

## Como rodar (desenvolvimento)
1. npm install
2. npm run dev
Servidor vai rodar em http://localhost:3333

## Endpoints
- GET /pokemons
- GET /pokemons/:id
- POST /pokemons  { name, type, height, weight, description }
- PUT /pokemons/:id
- DELETE /pokemons/:id
